# Open APIs

깃플 플랫폼의 자원을 REST API 형태로 직접 호출할 수 있습니다.

* API의 세부 규격은 [API Spec Document](http://guide.gitple.io/api)에 나와있습니다.


각 API를 호출하기 위해서는 아래 내용이 필요합니다.

* 앱코드(`app-code`)
* 오픈 API 시크릿(`secret`)
  - 워크스페이스 "**설정 > 연동**" 메뉴에서 값 확인 (처음일 경우 생성)
